# My Portfolio

A React + Vite portfolio website that showcases my web development projects, skills, and contact information.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal to preview the site.

## Build for production

```bash
npm run build
```

## Notes

- The project uses React and Vite for fast local development.
- The homepage includes sections for About Me, Projects, and a contact/message area.
